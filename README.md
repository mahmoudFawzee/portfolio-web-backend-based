# Portfolio Frontend

A modern, responsive web frontend for the Portfolio API built with Vanilla JS and Vite.

## Prerequisites

- Node.js (v16 or higher)
- Backend running at `http://localhost:5001`

## Setup & Running

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open in browser**:
   The application will be available at `http://localhost:5173` (or the port shown in your terminal).

## Features

- **Modern UI**: Clean design with glassmorphism and smooth animations.
- **Responsive**: Works perfectly on mobile and desktop.
- **Dynamic Data**: Fetches real-time profile data from the backend API.
- **Fast**: Powered by Vite for an ultra-fast development experience.
